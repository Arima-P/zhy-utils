# @fatyu/zip

> compress resource cli

## 能力

- 压缩文件、文件夹为 zip 压缩包
- 支持加密压缩

## 快速开始

### 安装

```sh
# npm
npm i @fatyu/zip -D

# yarn
yarn add @fatyu/zip -D
```

### 使用

```sh
# 简单使用

  # npm
  npm zip -p /relative-path/to/target

  # yarn
  yarn zip -p /relative-path/to/target

# 更多例子（yarn 为例)

  # 帮助
  yarn zip -h
```
